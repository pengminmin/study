## 其它

1. `git push -u origin master` 中的 `-u` 的作用

   `-u` 的作用是指定默认推送分支，以后输入 `git push` 就是后面设置的 `origin master`

2. `ssh-keygen -t rsa -C "540965717@qq.com"` 中的 `-t` 和 `-C` 的作用

   `-t` 是设置密钥类型，这里设置的密钥类型为 `rsa` ，`-C` 用于识别这个密钥的注释，在密钥最后面会有注释的内容，这里设置的注释是 `540965717@qq.com`

3. 重新加载配置文件

   ```bash
   composer dump-autoload
   ```

4. npm 设置代理

   ```bash
   # 设置代理
   npm config set proxy http://127.0.0.1:1080
   # 设置源
   npm config set registry=http://registry.npmjs.org
   # 删除代理
   npm config delete proxy
   # 源设置回来
   npm config set registry=https://registry.npm.taobao.org/
   ```
