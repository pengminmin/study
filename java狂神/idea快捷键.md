1. Alt + Insert 代码自动生成，如生成对象的 set / get 方法，构造函数，toString() 等 （必备）
2. Ctrl + Shift + Enter 自动结束代码，行末自动添加分号 （必备)
3. Ctrl + Shift + F12 编辑器最大化 （必备）
4. Ctrl + Alt + L 格式化代码，可以对当前文件和整个包目录使用 （必备）
5. Ctrl + Alt + O 优化导入的类，可以对当前文件和整个包目录使用 （必备）
